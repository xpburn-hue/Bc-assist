# BC Assist

BC Assist is a ballistic coefficient estimation and trajectory analysis project.

The goal is to build a tool that combines ballistic modeling with measured field data to improve BC estimates and trajectory predictions.

## Planned Features

- Ballistic trajectory calculations
- Environmental correction inputs
- BC estimation from measured data
- Time-of-flight analysis
- Sensor integration
- Confidence scoring

## Long Term Vision

Support a field-capable workflow using:

- Chronograph data
- Environmental sensors
- Acoustic detection of muzzle report and impact
- Camera assisted confirmation
- Mobile or embedded hardware interfaces

See `docs/project-plan.md` and `docs/system-design.md` for current design notes.
